# ⛽ Дизель Монітор UA

Дашборд цін на дизельне пальне в Україні у стилі фінансового термінала.
Візуальна мова успадкована від [vpn-speedtest](https://github.com/Ros-ua/vpn-speedtest) (Pro Speedtest).

**Живий сайт:** https://diesel-monitor.pp.ua/

## Що показує

- **Поточна середня ціна ДП** по Україні + зміни за день / тиждень / місяць / рік
- **Інтерактивний графік** історії цін (7д…весь час, зум/пан, всі види пального)
- **Порівняння мереж АЗС** (36+ мереж): ціна, динаміка, відхилення від середньої, сортування, пошук, фільтри
- **Сторінки мереж і областей** — історія, статистика, ціни по регіонах
- **Чому змінюється ціна** — автоматичний аналіз чинників (Brent, курс USD/UAH, динаміка, новини)
- **Аналітика** — медіана, волатильність, екстремуми, середні за місяць/рік
- **Прогноз тренду** (зважена регресія; оцінка, не гарантія)
- **Сповіщення** — правила типу «ДП нижче 55 грн» (localStorage + Notification API)
- **Новини ринку** з індикатором впливу на ціну 🔴🟡🟢

## Дані

| Джерело | Що береться |
|---|---|
| [Мінфін](https://index.minfin.com.ua/ua/markets/fuel/) (дані Консалтингової групи А-95) | середні ціни, ціни мереж по областях |
| Wayback Machine | реальна історія середніх цін з 2024 року (206+ днів) |
| [НБУ](https://bank.gov.ua) | офіційний курс USD/UAH |
| Yahoo Finance | Brent (BZ=F), денні закриття |
| Економічна правда, УНІАН, Укрінформ, РБК-Україна (RSS) | новини, автофільтр за темою пального |

Історія **накопичується автоматично**: GitHub Actions двічі на день запускає
`scripts/collect.mjs`, комітить оновлені `public/data/*.json` і передеплоює сайт.
Національна ціна мережі = медіана по областях присутності.

## Стек

React 19 · TypeScript (strict) · Vite · Tailwind CSS v4 · Apache ECharts · Framer Motion ·
react-router (hash) · oxlint + Prettier. Бекенд не потрібен — статичні JSON + Actions cron.

## Розробка

```bash
npm install
npm run dev        # локальний дев-сервер
npm run build      # прод-збірка у dist/
npm run collect    # разовий збір поточних цін
npm run backfill   # бекфіл історії (Wayback + НБУ + Brent), інкрементний
```

Додати новий вид пального: ключі `FuelKey` у `src/types.ts` вже покривають
А-95/А-92/газ — дані збираються для всіх, достатньо UI-перемикача.
Замінити джерело даних: реалізувати інтерфейс `DataProvider` у `src/api/provider.ts`.

## Застереження

Прогнози — статистичні оцінки за історичними даними, **не фінансова порада**.
Ціни носять довідковий характер; актуальні ціни уточнюйте на АЗС.
